# STRETCH

棒を伸ばして となりの足場へ渡る、ワンタップのカジュアルゲーム。
スマホ・PCのブラウザで動く単一の静的Webアプリ（ビルド不要）。

## 遊び方
- 画面を**押している間**、棒が伸びる
- 指を**はなす**と棒が倒れて橋になる
- 次の足場に届けば前進。短すぎ・伸ばしすぎは落下でゲームオーバー
- 足場中央の**パーフェクトゾーン**に倒すとコンボ倍率アップ

## 技術
- HTML5 Canvas 2D + Vanilla JavaScript + Web Audio API
- 外部ライブラリ・ビルド不要。`index.html` を開くだけで動作

## ローカルで動かす
`index.html` をブラウザで開くだけ。
